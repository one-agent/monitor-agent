# 胜算云平台知识库

## 平台简介

## 平台功能

### AI 模型调用
- 支持多种主流大语言模型
- 包括 GPT-4、通义千问、文心一言等
- 提供 API 和 SDK 两种调用方式
- 支持流式输出和函数调用

### 知识库管理 (RAG)
- 支持上传 Markdown、PDF、Word 等格式文档
- 自动文档切片和向量化
- 支持本地知识库和云端知识库
- 语义检索，精准匹配相关内容

### 监控告警
- 实时监控 API 调用状态
- 自动检测异常和超时
- 支持多种告警方式（邮件、飞书、钉钉）
- 告警可自定义阈值和频率

### API 管理与测试
- 提供在线 API 文档
- 支持在线调试和测试
- 提供调用统计和分析报表
- 支持 API Key 管理

## 开发指南

### 快速开始

1. 注册账号并获取 API Key
2. 选择合适的模型
3. 参考 API 文档进行调用
4. 在控制台监控调用情况

### API 调用示例

```bash
curl -X POST https://api.shengsuan.com/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4",
    "messages": [
      {"role": "user", "content": "你好"}
    ]
  }'
```

### SDK 支持

- Python SDK: `pip install shengsuan-python`
- Java SDK: Maven 依赖 `com.shengsuan:shengsuan-java`
- Node.js SDK: `npm install shengsuan-js`

## 常见问题 (FAQ)

### Q: 如何选择合适的计费模式？
A: 如果您的调用量不稳定或较少，建议使用按量付费；如果有稳定的高调用量，包月订阅更划算。

### Q: API 调用失败怎么处理？
A: 请检查 API Key 是否正确，查看控制台的调用日志。如问题持续存在，请联系技术支持。

### Q: 支持私有化部署吗？
A: 企业版客户支持私有化部署，请联系销售团队获取报价。

### Q: 如何设置告警通知？
A: 在监控配置页面添加您的告警方式（如飞书 Webhook），设置阈值即可。

### Q: 知识库有什么限制？
A: 单个知识库最多支持 10000 个文档，每个文档最大 10MB。企业版支持更高限制。

### Q: 支持哪些模型？
A: 目前支持 GPT-4/GPT-3.5、通义千问系列、文心一言系列、Claude 系列等主流模型。
